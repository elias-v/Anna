# Anna System – Command Index

## Kurzbefehle (mit führendem Punkt)

| Befehl | Beschreibung |
|---|---|
| `.team` | Zeige alle Team-Mitglieder und ihre Aufgaben |
| `.hilfe` | Liste alle verfügbaren Kurzbefehle auf |
| `.funktionen` | Zeige die Fähigkeiten jedes Spezialisten |
| `.session-log` | Erkläre die verschiedenen Log-Typen (Session, Proactive, Realignment, Insight) |
| `.readme` | Zeige die README-Datei des Anna-Systems an |

## Wartung

- Diesen Index bei Änderungen an den Kurzbefehlen aktualisieren
- Neue Befehle hier eintragen, bevor sie in den Verträgen referenziert werden
